# 安装指南（请配合AI助手使用）

这个GPU加速的2D像素渲染器需要以下库来构建和运行。请按照以下步骤安装。

## 必需的库

### 1. GLFW (窗口和输入管理)

**下载链接**: https://www.glfw.org/download.html

**安装步骤**:
1. 下载 Windows 预编译二进制文件 (32-bit 或 64-bit，根据您的系统)
2. 解压到您选择的目录（例如 `C:\libs\glfw`）
3. 在解压的目录中，您会找到：
   - `include/` - 头文件
   - `lib-vc2022/` 或类似的 - 库文件

### 2. GLEW (OpenGL扩展加载)

**下载链接**: https://glew.sourceforge.net/

**安装步骤**:
1. 下载 Windows 32-bit 或 64-bit 二进制文件
2. 解压到目录（例如 `C:\libs\glew`）
3. 目录中包含：
   - `include/` - 头文件
   - `lib/Release/x64/` - 64位库文件
   - `bin/Release/x64/` - 运行时DLL

### 3. GLM (OpenGL数学库)

**下载链接**: https://github.com/g-truc/glm/releases

**安装步骤**:
1. 下载最新版本的源码
2. 解压到目录（例如 `C:\libs\glm`）
3. 这是一个只有头文件的库，只需要 `glm/` 目录

## 配置项目

### 选项A: 手动配置（推荐）

1. 将上述库的 `include/` 目录路径添加到项目的包含路径
2. 将库文件（`.lib`）添加到链接器输入
3. 将DLL文件复制到可执行文件目录

### 选项B: 使用CMake自动下载（如果网络允许）

项目已经配置了CMake来自动下载这些库，但如果网络有问题或需要离线构建，请使用手动配置。

## 验证安装

安装完成后，请告诉我以下信息：

1. 每个库的安装路径（例如：`C:\libs\glfw`）
2. 您的系统架构（32位或64位）
3. 您希望使用哪个编译器（MSVC或MinGW）

## 故障排除

### 常见问题

1. **CMake找不到库**: 手动设置 `GLFW_ROOT`、`GLEW_ROOT`、`GLM_ROOT` 环境变量
2. **链接错误**: 确保库文件的架构（32/64位）与项目设置匹配
3. **运行时缺少DLL**: 将GLEW的DLL复制到可执行文件目录

### 快速测试

您可以运行以下命令测试OpenGL是否正常工作：

```bash
# 检查OpenGL版本
glxinfo | grep "OpenGL version"  # Linux
# 或使用专门的OpenGL检测工具
```
